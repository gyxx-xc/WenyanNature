# Wenyan Judou Project

## Overview
Wenyan Judou is a Java-based interpreter and compiler for the Wenyan programming language, designed to execute classical Chinese-style code constructs. This project provides a runtime environment, bytecode generation, and execution capabilities for Wenyan programs.

## Modules
- `antlr`: ANTLR grammar and error handling.
- `compiler`: Bytecode generation and compilation logic.
- `exec_interface`: Execution interfaces and request handling.
- `runtime`: Core runtime environment and thread management.
- `structure`: Data structures and value representations.
- `utils`: Utility classes for language processing and translation.
