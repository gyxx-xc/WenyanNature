# 符咒

> 符者，三光之灵文，天真之信也。 ==《秘要诀法》==

!!! note

    由 AI 学习源码后生成（未检查）

用于运行文言编程代码的核心方块，支持 7 个不同等级。

## 使用

### 运行代码

1. **放置符咒** - 将符咒放置在平面上
2. **编辑代码** - **潜行右键**打开代码编辑界面，输入文言编程代码
3. **执行运行** - **直接右键**运行代码

!!! tip 提示

    - 按住 Shift 并右键可**打开代码编辑界面**
    - 运行时会根据代码执行状态显示不同颜色和亮度的光效
    - 代码错误时会显示红色警告
    - 在代码编辑界面中，按住 Shift 可查看代码片段的详细信息

### 代码编辑

在代码编辑界面中：

- 输入文言编程语法编写的代码
- 代码会自动保存
- 支持导入附近其他符咒或设备的代码
- 可为符咒命名（显示时会添加「」括号）

## 等级系统

符咒分为 7 个等级，等级越高运行速度越快：

| 等级  | 名称   | 运行速度 (步/刻) |
|-----|------|------------|
| 一阶符 | 一阶符 | 1          |
| 二阶符 | 二阶符 | 10         |
| 三阶符 | 三阶符 | 100        |
| 四阶符 | 四阶符 | 1,000      |
| 五阶符 | 五阶符 | 10,000     |
| 六阶符 | 六阶符 | 100,000    |
| 七阶符 | 七阶符 | 1,000,000  |

!!! warning 注意

    高等级符咒需要相应的材料制作，请参考合成表获取。

## 状态显示

符咒会通过颜色和亮度显示当前运行状态：

- **不运行** - 灰色，无光源
- **运行中** - 白色，亮度 4
- **空闲等待** - 黄色，亮度 4
- **错误状态** - 红色，亮度 10

## 交互功能

### 与实体交互

符咒可以作为"平台"被其他方块设备调用：

- 附近的通讯模块可以导入此平台的代码
- 支持跨平台函数调用
- 最远支持 3 格距离的设备通信

### 数据输出

运行时产生的输出会显示在：

- 客户端标题栏（实时显示）
- 代码编辑界面的输出窗口可查看完整输出历史
- 使用「書」函数可在代码中输出文本

## 获取方式

通过手持符咒物品放置获得对应等级的符咒方块。

!!! tip 小知识

    - 符咒在运行时会消耗极少的能量
    - 破坏符咒方块会掉落对应的手持符咒物品
    - 代码会保存在符咒中，即使破坏也会保留
