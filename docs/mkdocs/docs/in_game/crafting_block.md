# 创石

!!! note

    由 AI 学习源码后生成（未检查）

用于**解答编程问题**来制作物品的核心方块

## 用途

创石是一种特殊的合成装置，需要通过**编写 Wenyan 程序**来解答问题以完成物品制作。与普通的合成台不同，它需要你编写正确的程序来解决问题，才能成功制作出目标物品。

## 使用方法

### 基础设置

1. **放置创石**：将创石放置在合适的位置
2. **布置基石**：在创石周围**3格范围内**放置**基石**（Pedestal），用于放置材料
3. **放入材料**：在基石上放入所需的材料

### 开始制作

1. 与创石交互打开编程界面
2. 使用 `「参」` 函数获取输入变量（如 `「甲」`、`「乙」` 等）
3. 编写 Wenyan 程序计算正确答案
4. 使用 `書` 函数提交答案

### 编程规则

- 每个配方有特定的**问题类型**（如 "plus"、"echo"、"print" 等）
- 每个配方需要完成指定**轮数**（round）的正确回答
- 答错时当前轮次进度会**清零**，需要重新开始该轮
- 成功时会有**文字粒子效果**显示你的答案
- 全部轮次答对后，制作完成的物品会出现在创石上方

## 注意事项

- 创石会**搜索周围 3×3×3 立方体范围内**的基石来获取材料
- 制作过程中请确保基石上的材料**不会被移走**
- 不同的配方有不同的**问题类型**和**所需轮数**
- 配方通过 JSON 文件定义，包含输入材料、输出物品、问题类型和轮数
