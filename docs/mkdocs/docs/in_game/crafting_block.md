# 创石

!!! note

用于**解答问题**来制作物品的核心方块

## 用途

创石是一种特殊的合成装置，需要通过**解答问题**来完成物品制作。与普通的合成台不同，它需要你理解并正确回答题目，才能成功制作出目标物品。

## 使用方法

### 基础设置

1. **放置创石**：将创石放置在合适的位置
2. **布置基石**：在创石周围放置**基石**（Pedestal），用于放置材料
3. **放入材料**：在基石上放入所需的材料

### 开始制作

1. **右键点击**创石打开操作界面
2. 查看需要解答的**题目**
3. 使用文言编程语言编写答案
4. 点击**「书」**提交答案

### 答题规则

- 每道题可能需要**多次正确答案**才能完成制作
- 答错时进度会**清零**，需要重新开始
- 成功时会有**视觉特效**显示你的答案
- 全部答对后，制作完成的物品会出现在创石上方

## 注意事项

- 创石会**搜索周围 3 格内**的基石来获取材料
- 制作过程中请确保基石上的材料**不会被移走**
- 不同的配方有不同的**难度等级**和**所需轮数**
- 答案提交时会消耗一定的**耐久度**

## 技巧提示

- 观察基石上的材料组合，确定要制作的物品
- 仔细阅读题目要求，理解后再作答
- 如果答错了不要气馁，重新思考后再次尝试
- 成功的答案会产生漂亮的**文字粒子效果**
