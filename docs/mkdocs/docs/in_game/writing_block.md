刻印台是文言编程中最重要的多方块结构核心组件，用于将制作完成的符咒进行镌刻激活，使其获得真正的法力。

## 用途

刻印台的主要作用是对**一阶符至六阶符**进行镌刻。未经镌刻的符咒只是普通的物品，只有通过刻印台的处理，才能成为真正可用的法术载体。

### 工作流程

1. **放置符咒**：手持已制作好的符咒（一阶符至六阶符），右键点击刻印台，将符咒放入其中
2. **打开界面**：潜行状态下（按住 Shift）右键刻印台，打开镌刻界面
3. **编写代码**：在打开的界面中编写文言编程代码
4. **保存镌刻**：完成编写后关闭窗口，代码会自动镌刻到符咒上
5. **取出成品**：再次右键刻印台，取出已镌刻完成的符咒

## 使用技巧

### 界面操作

- **正常右键**：放入或取出符咒物品
- **潜行右键**：打开镌刻编辑界面
- **标题命名**：可以在界面顶部为当前符咒命名，方便识别
- **代码片段**：右侧提供常用代码片段，点击即可插入

### 符咒存放

- 刻印台可以存储最多 64 个同种符咒
- 支持批量镌刻：放入多个相同符咒时，只需编写一次代码
- 镌刻过程中可以随时中断，进度不会丢失

### 视觉提示

- 刻印台顶部会根据内部符咒数量显示不同高度的悬浮效果
- 符咒数量越多，悬浮层次越高（每 64 个符咒增加一层）

## 进阶用法

### 批量生产

当需要大量相同的符咒时，可以先在刻印台中放入多张符咒纸，进行一次镌刻后，所有符咒都会拥有相同的代码。这对于制作常用的基础法术非常有用。

### 命名管理

为不同的符咒起一个直观的名字非常重要。好的命名可以让你在后续使用时快速找到需要的符咒，避免混淆。
