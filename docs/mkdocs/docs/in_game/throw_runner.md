# 投掷符咒

> 朱书黄素，烧灰，吹布十方，千真万灵俱集坛所。 ==《上清灵宝大法》==

!!! note

    由 AI 学习源码后生成（未检查）

可以扔出去的符咒，扔出去后会在空中飞行并运行代码。

## 使用

### 投掷符咒

1. **拿在手上** - 在物品栏中选择投掷符咒
2. **右键扔出** - 右键将符咒扔出去
3. **空中运行** - 符咒会在空中飞行并执行代码
4. **自动消失** - 运行结束后符咒会自动消失

!!! tip 提示

    - 扔符咒后有 1 秒冷却时间
    - 扔出去会消耗 1 个符咒
    - 符咒在空中会慢慢减速
    - 符咒最多在空中存在 5 秒

### 代码执行

扔出去的符咒会运行里面存储的代码：

- **加载代码** - 扔出去时读取符咒里的代码
- **创建环境** - 创建一个独立的运行空间
- **加载模块** - 加载符咒里携带的附加功能（符）
- **错误提示** - 出错时会告诉扔符咒的人

### 模块系统

投掷符咒可以携带附加功能：

- **存储模块** - 最多可以带 256 个附加功能
- **自动加载** - 扔出去时所有功能都会生效
- **避免重复** - 相同名字的功能只加载第一个
- **一次性使用** - 功能用完后会消失

## 飞行特性

### 飞行特点

- **飘在空中** - 符咒在空中不会掉下来
- **碰到就停** - 撞到方块会停下来
- **不能捡回** - 飞在空中的符咒不能捡起来
- **带火效果** - 飞行时看起来像在燃烧

### 飞行轨迹

- **快速飞出** - 扔出去时速度很快
- **旋转飞出** - 会旋转着飞出去
- **慢慢减速** - 飞得越远速度越慢
- **自动转向** - 飞行时会自己调整方向
