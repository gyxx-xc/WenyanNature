## 运算符优先级

从高到低：
1. `變` (一元)
2. `乘` `除`
3. `加` `減`
4. `等於` `不等於` `大於` `小於` `不大於` `不小於`
5. `且` `或` `中無陰乎` `中有陽乎`

## 介词用法

- `以` - 第二个操作数（标准顺序）
- `於` - 第一个操作数（反转顺序）

示例：
```wenyan
加一以二 = 1 + 2
加一於二 = 2 + 1
```

| `之` | 引用上一个结果 | 否 |
| `己` | 当前对象实例 | - |

## 常见错误

### ❌ 错误：混用其和之
```wenyan
除其以其 // 未定义行为
```

### ✅ 正确：使用之累积
```wenyan
夫一。夫之。夫之。取三以加。
```

### ❌ 错误：忘记类型转换
```wenyan
加「「三」」以一 // 可能出错
```

### ✅ 正确：显式转换
```wenyan
夫「「三」」。加其一。 // 先转为数字
```

3. 减少不必要的类型转换
