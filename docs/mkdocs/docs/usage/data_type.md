# 数据类型

## 數 数字
用于表示数值，支持整数、负数和小数。

```
一万一千零五 11005
一一四五一四 114514
負一又二分三毫 -1.203
```

## 言 字符串
用于表示文本内容，由双引号包裹的字符序列组成。

```
「「xxx」」 "xxx"
```

## 爻 布尔值
表示逻辑真或假的值。

```
陰 false
陽 true
```
