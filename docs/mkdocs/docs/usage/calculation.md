# 算術第三

本文介绍基本算术运算与变量操作之法。凡计算之事，皆可以此法推演。
```
加一以二。書之。 批曰「「 三。」」
減二以一。書之。 批曰「「 一。」」
乘二以三。書之。 批曰「「 六。」」
除八以四。書之。 批曰「「 二。」」
除五以三。所餘幾何。書之。 批曰「「 二。」」
```

另有补充语法：
```
且陰以陽。書之。 批曰「「 陰。」」
或陽以陰。書之。 批曰「「 陽。」」
變陰。書之。 批曰「「 陽。」」
```

前置词可用「於」或「以」，二者将改变运算中数字之顺序。例如：
```
減二以一。書之。 批曰「「 一。」」
減二於一。書之。 批曰「「 負一。」」
```
欲延长计算式，可借代词（其）为之：
```
加一以二。乘其以三。書之。 批曰「「 (1+2)*3 = 九。」」
```
更复杂之例：
```
吾有二數。曰三十五。曰九十四。名之曰「頭」曰「足」。
除「足」以二。減其以「頭」。名之曰「兔」。
減「頭」以「兔」。名之曰「雉」。
夫「雉」夫「兔」。取二以書。
```

数据会按需转换为对应类型，例如：
```
或陰以三。書之。 批曰「「 陽。」」
加二以陽。書之。 批曰「「 三。」」
乘三以陰。書之。 批曰「「 零。」」
加二以「「者」」。書之。 批曰「「 「「二者」」。」」
```
