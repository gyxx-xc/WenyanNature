## 导入语法

导入语句用于引入外部模块或包。有三种导入语法：

### 基本导入

导入整个包：
```wenyan
吾嘗觀「甲」之書。
```

等价于现代编程语言中的：
```python
import 甲
```

### 选择性导入

仅导入包中的特定属性或函数：
```wenyan
吾嘗觀「甲」之書。方悟「乙」「丙」之義。
```

等价于：
```python
from 甲 import 乙, 丙
```

### 导入并命名

导入包后为其指定别名：
```wenyan
吾嘗觀「甲」之書。名之曰「乙」。
```

等价于：
```python
import 甲 as 乙 
```

导入的包可以作为后续操作的基础，例如访问其属性或调用其中的函数。
