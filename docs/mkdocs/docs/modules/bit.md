# 位元符

!!! note
    由AI学习源码后生成（未检查）

位运算模块，提供二进制位操作功能

## 提供

模块名：「位經」

- 「左移」
    - 参数：二數
    - 描述：将第一个数向左移动第二个数指定的位数
    - 返回：一數
- 「右移」
    - 参数：二數
    - 描述：将第一个数向右移动第二个数指定的位数（保留符号位）
    - 返回：一數
- 「補零右移」
    - 参数：二數
    - 描述：将第一个数向右移动第二个数指定的位数（高位补零）
    - 返回：一數
- 「位與」
    - 参数：二數
    - 描述：对两个数进行按位与运算
    - 返回：一數
- 「位或」
    - 参数：二數
    - 描述：对两个数进行按位或运算
    - 返回：一數
- 「異或」
    - 参数：二數
    - 描述：对两个数进行按位异或运算
    - 返回：一數
- 「與非」
    - 参数：二數
    - 描述：对两个数进行与非运算
    - 返回：一數
- 「位變」
    - 参数：一數
    - 描述：对数进行按位取反运算
    - 返回：一數

## 补充说明

所有运算均为整数位运算，适用于需要精确控制二进制位的场景。
