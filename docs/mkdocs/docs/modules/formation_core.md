# 阵眼

!!! note
    由 AI 学习源码后生成（未检查）

符咒控制模块，提供对「可戴符」的启动、状态查询和批量管理能力

## 提供

模块名：「眼」

- 「啓」
    - 参数：若干言
    - 描述：启动指定名称的符
        - 每个参数为符的名称（不含引号）
        - 会自动建立与目标符咒的通信连接
        - 若找不到指定的可戴符则抛出异常
    - 返回：无
- 「狀」
    - 参数：一言
    - 描述：查询指定可戴符的运行状态
        - 参数为可戴符的名称
        - 若找不到则抛出异常
    - 返回：一運行類（见下方定义）
- 「歸」
    - 参数：无
    - 描述：等待所有已启动的可戴符执行完毕
        - 自动清理已完成和无效的可戴符引用
        - 阻塞当前线程直到所有符咒停止运行
    - 返回：无

## 運行類

运行状态类型，表示可戴符的执行状态：

- `running`：正在运行
- `idle`：空闲
- `error`：错误状态
- `not_running`：未运行
