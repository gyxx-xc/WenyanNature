# 活塞模組

!!! note
    由 AI 分析源码后生成（未检查）

活塞控制模块，提供可编程的活塞推拉功能

## 提供

模块名：「移」

- 「推」
    - 参数：
        - 一言：位置坐标（三维向量）
        - 一言：方向向量（三维向量）
    - 描述：推动指定位置的方块结构
        - 根据方向向量确定推动方向
        - 粘性方块会带动相邻方块
        - 无法推动的方块会被摧毁
    - 返回：无
    - 异常：
        - "Invalid direction"：方向向量无效
        - "Block already exists"：目标位置已有方块
        - "Failed to move block"：推动失败

- 「拉」
    - 参数：
        - 一言：位置坐标（三维向量）
        - 一言：方向向量（三维向量）
    - 描述：拉动指定位置的方块结构
        - 从相反方向拉动方块
        - 粘性方块会带动相邻方块
        - 拉动时会在原位置生成装饰性活塞头
    - 返回：无
    - 异常：同「推」

## 与其他模块的关系

- 依赖「向量符」模块提供三维向量参数
- 可与「方塊符」模块配合检测和放置方块
- 可被「實體符」模块的实体互动功能增强
- 与「天下情報符」的红石检测功能互补
