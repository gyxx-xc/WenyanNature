# 集符

!!! note
    由AI学习源码后生成（未检查）

集合操作模块，提供列表和集合的各种操作功能

## 提供

模块名：「集」

- 「無交」
    - 参数：二列
    - 描述：判断两个列表是否没有共同元素
    - 返回：一爻
- 「交集」
    - 参数：二列
    - 描述：求两个列表的交集
    - 返回：一列
- 「差集」
    - 参数：二列
    - 描述：求第一个列表相对于第二个列表的差集
    - 返回：一列
- 「反轉」
    - 参数：一列
    - 描述：反转列表中的元素顺序
    - 返回：一列
- 「排序」
    - 参数：一列
    - 描述：对列表进行排序（要求元素可比较）
    - 返回：一列
- 「包含」
    - 参数：一列、一物
    - 描述：判断列表是否包含指定元素
    - 返回：一爻
- 「最大」
    - 参数：一列
    - 描述：求列表中的最大元素
    - 返回：一物
- 「最小」
    - 参数：一列
    - 描述：求列表中的最小元素
    - 返回：一物

## 补充说明

- 列表中的元素必须是可比较的类型才能使用排序、最大值、最小值等功能
- 空列表的最大值和最小值返回零
