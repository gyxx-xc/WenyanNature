# 實體符

!!! note
    由AI学习源码后生成（未检查）

实体操作模块，提供世界中实体的检测和交互功能

## 提供

模块名：「实」

- 「察域之实」
    - 参数：二向量
    - 描述：检测指定区域内的所有实体
    - 返回：一列（實體列表）
- 「近域之实」
    - 参数：一分數（半径）
    - 描述：检测符咒周围指定半径内的所有实体
    - 返回：一列（實體列表）
- 「实之视」
    - 参数：二向量（起点、终点）
    - 描述：检测从起点到终点的视线上的方块（射线追踪）
    - 返回：一方塊

## 补充说明

- 实体检测会排除旁观者模式的玩家
- 「实之视」使用射线追踪技术，可用于检测玩家或生物的视线
- 返回的实体列表可以进一步处理和筛选
