# 爆裂符

!!! note
    由AI学习源码后生成（未检查）

爆炸和火焰效果模块，提供各种破坏性效果

## 提供

模块名：「爆」

- 「雷」
    - 参数：无
    - 描述：在符咒位置召唤闪电
    - 返回：无
- 「爆」
    - 参数：无
    - 描述：在符咒位置产生爆炸（半径 3.0）
    - 返回：无
- 「燃」
    - 参数：一向量（偏移）
    - 描述：点燃指定位置的方块
    - 返回：无
- 「火球」
    - 参数：一向量（速度）
    - 描述：发射一个大火球
    - 返回：无

## 补充说明

- 所有效果均在服务器端执行
- 爆炸会破坏方块并造成伤害
- 使用时需注意安全，避免误伤
