# 天下情報符

!!! note
    由AI学习源码后生成（未检查）

世界信息模块，提供红石信号和天气控制功能

## 提供

模块名：「信」

- 「量」
    - 参数：无
    - 描述：检测周围最强的红石信号强度
    - 返回：一數
- 「輸能」
    - 参数：一數
    - 描述：输出指定强度的红石信号
    - 返回：无
- 「觸」
    - 参数：无
    - 描述：触发更新（内部使用）
    - 返回：无
- 「變天」
    - 参数：一言
    - 描述：改变天气
        - 「晴」：设置为晴天
        - 「雨」：设置为雨天
        - 「雷」：设置为雷暴
    - 返回：无

## 补充说明

- 红石信号检测范围为相邻方块
- 天气改变仅在服务器世界生效
- 可作为红石电路的可编程替代方案
