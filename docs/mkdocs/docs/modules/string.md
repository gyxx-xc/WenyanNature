# 字串符

!!! note

字符串操作模块，提供文本处理功能

## 提供

模块名：「文」

- 「長」
    - 参数：一言
    - 描述：求字符串的长度
    - 返回：一數
- 「取」
    - 参数：一言、一數
    - 描述：从字符串中取出指定位置的字符
    - 返回：一言
- 「尋」
    - 参数：二言
    - 描述：在第一个字符串中查找第二个字符串的位置
    - 返回：一數
- 「分」
    - 参数：二言
    - 描述：用第二个字符串作为分隔符，分割第一个字符串
    - 返回：一列
- 「換」
    - 参数：三言
    - 描述：将第一个字符串中的第二个字符串替换为第三个字符串
    - 返回：一言
- 「反」
    - 参数：一言
    - 描述：反转字符串
    - 返回：一言
- 「去空」
    - 参数：一言
    - 描述：去除字符串首尾的空白字符
    - 返回：一言
- 「包含」
    - 参数：二言
    - 描述：判断第一个字符串是否包含第二个字符串
    - 返回：一爻
- 「起始為」
    - 参数：二言
    - 描述：判断第一个字符串是否以第二个字符串开头
    - 返回：一爻
- 「結束為」
    - 参数：二言
    - 描述：判断第一个字符串是否以第二个字符串结尾
    - 返回：一爻

## 补充说明

字符串索引从零开始计数。
