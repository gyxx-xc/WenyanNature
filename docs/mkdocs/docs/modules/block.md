# 方塊符

!!! note
    由AI学习源码后生成（未检查）

方块操作模块，提供 Minecraft 方块的检测和获取功能

## 提供

模块名：「塊」

- 「尋」
    - 参数：三物（起始向量、结束向量、方塊物品或方塊類）
    - 描述：在指定范围内搜索是否存在匹配的方块
    - 返回：一爻
- 「取」
    - 参数：一向量
    - 描述：获取指定位置的方块状态
    - 返回：一方塊
- 「附」
    - 参数：无
    - 描述：获取与此符相邻的方块
    - 返回：一方塊

## 补充说明

- 「尋」方法会高亮显示搜索范围，便于调试
- 向量坐标相对于符咒本身的位置
- 可用于自动化检测、建筑辅助等场景
