吾有一術（Wenyan Nature）是一款受到 [文言](https://github.com/wenyan-lang/wenyan) 啟發的 Minecraft 模組。
透過文言程式語言，此模組讓玩家能編寫複雜的文言邏輯，創造並實現神奇的魔法效果。

本模組仍在開發中。

---
## 功能特點

- **書寫符咒代碼**：可直接於符咒中輸入並執行代碼。
- **符力等級驅動**：程式執行速度將隨符咒等級而提升。
- **升級編程環境**：使用紙和墨水升級符咒，解鎖新功能和更快的執行速度。
- **連接麥塊世界**：透過介面與 Minecraft 互動，例如放置方塊、觸發紅石訊號。
