吾有一术（Wenyan Nature）是一款受到 [文言](https://github.com/wenyan-lang/wenyan) 启发的 Minecraft 模组。
通过文言编程语言，此模组让玩家能够编写复杂的魔法逻辑，创造和实现神奇的魔法效果。

该模组仍在开发中。

---
## 功能特点

- **书写符咒代码**：可直接在符咒中输入并执行代码。
- **符力等级驱动**：程序执行的速度随着符咒等级提升而加快。
- **升级编程环境**：使用纸和墨水升级符咒，解锁新功能和更快的执行速度。
- **连接麦块世界**：通过接口与 Minecraft 交互，例如放置方块、触发红石信号。
